Лабораторна робота №2: Наука про дані: підготовчий етап



Опис завдання

Ця робота присвячена збору даних за допомогою `urllib`, їх очищенню через `pandas` та базовому аналізу часових рядів (VHI індекси) та споживання електроенергії.



Інструкція із запуску

1\. Створіть віртуальне середовище: `python -m venv venv`

2\. Активуйте його: `.\\venv\\Scripts\\activate`

3\. Встановіть залежності: `pip install -r requirements.txt`

4\. Запустіть Jupyter Notebook: `jupyter notebook`
