## Опис завдання
Проєкт складається з двох основних частин:
1. **Аналіз VHI індексів**: Збір даних з сервера NOAA для всіх областей України, їх очищення, перекодування та фільтрація.
2. **Аналіз енергоспоживання**: Робота з великим набором даних Individual Household Electric Power Consumption, обробка пропущених значень та вимірювання часу виконання операцій.

## Системні вимоги
* **Interpreter**: Python 3.10+
* **ОС**: Windows 10/11, Linux або macOS
* **Основні бібліотеки**: `pandas`, `urllib3`, `jupyter`

## Структура папки
* `data/` — завантажені CSV файли (ігноруються Git).
* `lab_02_part1.ipynb` — код для роботи з VHI індексами.
* `requirements.txt` — список необхідних бібліотек.
* `.gitignore` — локальні правила ігнорування.
