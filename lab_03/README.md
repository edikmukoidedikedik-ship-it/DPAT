## Опис датасету

Датасет: Communities and Crime

Джерело:
https://archive.ics.uci.edu/ml/datasets/Communities+and+Crime

Опис:
Датасет містить соціально-економічні показники громад США та рівень злочинності.

Кількість об'єктів: 1994
Кількість ознак: 128

Цільова змінна:
ViolentCrimesPerPop — рівень насильницьких злочинів на населення

Основні ознаки:

medIncome — медіанний дохід
pctPopUnderPov — рівень бідності
PctUnemployed — безробіття
population — населення

ВАЖЛИВО:
усі числові значення нормовані в діапазоні 0–1
